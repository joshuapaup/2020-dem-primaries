What if the 2020 Democratic National Convention was held tomorrow?

This website is an analysis on the 2020 U.S. Democratic Primaries. We will analyze polling data collected from the FiveThirtyEight and make inferences as to which states are more competitive based off average polling from earlier this year, show which candidates are leading in some states, and summarize national support for the leading candidates since September.

What is the current state of the primaries nationally (as of December 2019)?


This map shows which candidate is leading in each state. The data was collected from FiveThirtyEight, and includes polling data conducted in specific states. States without any polling data were excluded. The leader of each state was calculated by taking the average of each candidates polling across the entire primary season. This was done to account for long term trends in support, and demonstrate which candidate has had consistent support in a given state. This gives a summary of which candidate each state has supported throughout the primary process. As we can see based off these results, former Vice President Joe Biden leads with a majority of states.

Which seats are competitive?


This map shows which states are competitive based off theleading candidate's polling average. It uses polling data from the FiveThirtyEight, and is calculated by taking an average of all polling for top candidates in each state and subtracting the average polling data for the second place candidate. States with larger lead poll values are least competitive, while states with smaller lead poll values are more competitive. As we can observe, several polls suggest that quite a few states have clear front runners while others are a little more competitive.

How have the top candidates changed in the last few months?


This line graph tracks the polling averages for the top four candidates over the last three months. The data was collected by FiveThirtyEight and includes major national polling data conducted since the early primary season. We took data from the past three months and calculated a polling average for Joe Biden, Elizabeth Warren, Bernie Sanders, and Pete Buttigieg for the months of September, October, and November. The graph demonstrates very recent trends in polling particularly a decline in support for Elizabeth Warren, and an increase in support for Pete Buttigieg that coincided with a decline in support for Joe Biden.
